# Death-Image
## Death Image je hack and slash hra

## Settings:
### W/Space - Jump
### A - Walk left
### D - Walk right
### T - Attack

## Ve hře jsou 3 nepřátelé
### Jindra - Labuťo kachna
### Tonička - Orel
### Samanta - Ryba

## Drops:
### Jindra - Notes (peníze)
### Tonička - Heal 
### Samanta - Damage upgrade
